About This Project

This guide is a portable, offline tool designed to assist with native seed harvesting and ecological restoration in the Estancia Basin of New Mexico.

Data & Photo Credits

This project stands on the shoulders of the wider botanical and citizen science community. We extend our deep gratitude to:

  • iNaturalist Contributors: For providing vital, geolocated living plant imagery under Creative Commons licenses.
  • Herbarium Curators & Collectors: For preserving and digitizing the historical botanical specimens used for morphological reference.
  • Local Biologists: For field-testing the harvest windows and processing safety protocols.

Built as an open-access Progressive Web App (PWA). Data is synced dynamically via GitHub Pages.
